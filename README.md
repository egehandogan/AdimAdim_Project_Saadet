# Adım Adım - Saadet Partisi Dijital Seferberlik Platformu

"Adım Adım", Saadet Partisi gönüllüleri için geliştirilmiş, oyunlaştırma (gamification) prensiplerine dayalı bir mikro-görev platformudur. Özellikle 40 yaş üstü kullanıcılar düşünülerek, yüksek erişilebilirlik ve sade bir kullanıcı deneyimi ile tasarlanmıştır.

## 🚀 Özellikler

- **Gönüllü Paneli:** Günlük görevleri takip etme, puan kazanma ve seviye atlama.
- **Görev Sistemi:** 
  - **Direkt Görevler:** Anında puan kazandıran dijital aksiyonlar.
  - **Kanıtlı Görevler:** Fotoğraf veya belge yükleyerek 2X puan kazandıran saha görevleri.
- **Ödül Marketi:** Kazanılan puanlarla özel parti ürünleri ve koleksiyon parçaları sipariş etme.
- **Referans Sistemi:** Yeni üye davet ederek, onların kazançlarından %3 bonus kazanma.
- **Yönetici Paneli:** Görev yönetimi, kanıt onayı ve sipariş takibi için kapsamlı yönetim arayüzü.

## 🛠️ Teknoloji Yığını

- **Frontend:** React (Vite) + TypeScript
- **Styling:** Tailwind CSS (Özel tasarım jetonları ile)
- **Kimlik Doğrulama:** Clerk Auth
- **İkonlar:** Lucide-React
- **Yönetim:** TanStack Query (React Query)

## 📦 Kurulum

Projeyi yerel ortamınızda çalıştırmak için:

```bash
# Depoyu klonlayın
git clone https://github.com/egehandogan/AdimAdim_Project_Saadet.git

# Proje dizinine gidin
cd AdimAdim_Project_Saadet

# Bağımlılıkları yükleyin
npm install

# Geliştirme sunucusunu başlatın
npm run dev
```
